Sobre o Article Summarizer Agent

Plataforma backend para extração e sumarização assíncrona de artigos públicos da web.

O que o sistema faz

O sistema recebe uma URL, executa um pipeline de scraping com proteção SSRF, processa o texto com NLTK/langdetect, gera um resumo por TF-IDF ou Google Gemini, persiste o histórico da tarefa e disponibiliza artefatos para download.

A aplicação segue um desenho de monólito modular com separação entre domínio, aplicação, infraestrutura e apresentação, mantendo o pipeline de NLP desacoplado das rotas Flask.

Arquitetura implementada

  • Clean Architecture incremental
  • CQRS com handlers de command/query
  • Eventos de domínio para submissão e conclusão
  • Persistência com SQLAlchemy + Alembic
  • Auth persistida com usuários e RBAC
  • Cache coerente via backend compartilhado
  • Observabilidade com health e Prometheus
  • Execução assíncrona com Celery/Redis ou fallback local

Stack real do projeto

Python Flask SQLAlchemy Alembic PostgreSQL Redis Celery scikit-learn NLTK Google Gemini Prometheus Docker
Hardening
  • SSRF guard com bloqueio de redes privadas
  • Rate limiting por perfil de rota
  • JWT com CSRF para cookies
  • Rotação de segredo com `kid`
  • Audit log para eventos administrativos
Fluxo assíncrono
  1. URL submetida para API
  2. Tarefa persistida no banco
  3. Evento `TaskSubmitted` publicado
  4. Dispatcher envia para Celery/thread
  5. Pipeline processa e atualiza a tarefa
